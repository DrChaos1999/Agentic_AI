# Data

The repository does not redistribute the original Magnetic Tile images because the author repository does not expose a standard licence file. Obtain the dataset directly from its author-published GitHub repository:

```bash
python scripts/download_dataset.py
python scripts/validate_dataset.py
python scripts/prepare_dataset.py
```

`data/demo/` contains clearly labelled synthetic smoke-test images generated by this project. They are only for testing the API and FAISS workflow; they are not suitable for model evaluation.
